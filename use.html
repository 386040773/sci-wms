Server Index

A list of the datasets you have added to sci-wms are available at the root url of your sci-wms server. This page lists the unique dataset ID's, path to netcdf file (or OpenDAP endpoint), and whether or not the server will be updating the available data going forward (as you would for a forecast collection).

If your datasets are local netcdf files, the full path will be listed on this page. If this path is sensitive information in someway, then this is a problem. We are working to come up with a solution to this problem.

getCapabilities

http://server:port/wms/your_unique_datasetid/?request=getCapabilities

getMap

http://server:port/wms/your_unique_datasetid/?request=getMap&...

Sci-wms requires elevation and time parameters which are often not inlcuded in other wms servers.
